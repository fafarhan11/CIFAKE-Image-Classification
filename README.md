# CIFAKE Image Classification

This project classifies images as Real or AI-Generated using a deep learning CNN model.

## Project Title

CIFAKE: Image Classification and Explainable Identification of AI-Generated Synthetic Images

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Google Colab
- Kaggle Dataset

## Dataset

CIFAKE: Real and AI-Generated Synthetic Images

## Features

- Loads CIFAKE dataset
- Trains CNN model
- Evaluates model accuracy
- Predicts whether an image is Real or Fake
- Shows sample image prediction

## How To Run

1. Open the notebook in Google Colab.
2. Upload your Kaggle API key when asked.
3. Run all cells one by one.
4. The model will train and show prediction results.

## Output

The model predicts whether an input image is Real or AI-Generated.
